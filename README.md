# MentaCare App

## 🚀 Configuración

1️⃣ Instala dependencias:
```
npm install
```

2️⃣ Corre el proyecto:
```
npm start
```

## 🔗 Rutas
- `/` Login
- `/chat` ChatBot
- `/mood` MoodCheck
- `/summary` Resumen

## 🔥 Backend
- Local: usa `services/localService.js`
- Firebase: configura `services/firebase.js` con tus credenciales
